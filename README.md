Sistema de Gestão Local (SGL)
Descrição do Projeto

O Sistema de Gestão Local (SGL) é uma aplicação desenvolvida em Python com o objetivo de realizar o gerenciamento de produtos e movimentações de estoque por meio de um menu interativo no console.

O sistema permite cadastrar produtos, consultar informações, buscar produtos por nome, registrar entradas e saídas de estoque, gerar relatórios de movimentações e emitir relatórios gerenciais com estatísticas dos produtos cadastrados.

Os dados são armazenados em arquivos JSON, garantindo a persistência das informações mesmo após o encerramento do programa.

Funcionalidades
Cadastro de produtos.
Listagem de produtos.
Busca de produtos por nome.
Registro de entrada de estoque.
Registro de saída de estoque.
Relatório de movimentações.
Relatório gerencial.
Salvamento automático dos dados em arquivos JSON.
